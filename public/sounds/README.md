# Draw sound effects

Add these MP3 files for the shaking container draw experience:

- **shake.mp3** - Short rattle loop (maraca/dice rattle). Suggested: [Freesound.org](https://freesound.org) search "maraca" or "dice rattle"
- **lid-open.mp3** - Pop/unscrew (bottle cap, container open). Suggested: "bottle open", "cap twist"
- **winner.mp3** - Short fanfare/celebration

Ensure license allows commercial use (CC0, CC-BY, or similar). Place files in this directory.
If files are missing, the app will run without sound.
